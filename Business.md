Business Direction & Strategic Vision
Project Name: KhunQuant
Vision: To democratize sophisticated quantitative portfolio management for the Thai community through an open-source, agentic framework that bridges the gap between traditional Thai equity markets and global digital assets.

1. Core Value Proposition
Unified Execution: A single "brain" (orchestrator) that can execute trades on Bitkub and Binance while simultaneously monitoring SET (Stock Exchange of Thailand) via Settrade.

Privacy-First Automation: By leveraging the KhunQuant foundation, KhunQuant prioritizes local execution of logic, ensuring API keys and sensitive strategy parameters remain under the user's control.

Thai Market Specialization: Unlike global tools, KhunQuant is built with native support for Thai-specific brokerage nuances (Streaming, InnovestX, Dime) and local regulatory contexts.

2. Target Ecosystem
Equities & Mutual Funds (TH): Settrade (Streaming), InnovestX, Dime (for US Stocks/Thai Funds).

Digital Assets: Bitkub (Local leader), Binance (Global liquidity).

Technical Indicators: Integration with TradingView webhooks and custom Python-based alpha strategies.

3. Strategic Roadmap
Phase 1 (Foundation): Establish the "Interface Adapter" layer. Since different exchanges use different protocols (REST, WebSocket, FIX), KhunQuant will provide a unified schema.

Phase 2 (Orchestration): Implement the LLM Agent layer (KhunQuant) to interpret natural language commands like "Rebalance my crypto into Thai high-dividend stocks if Bitcoin hits a certain RSI."

Phase 3 (Community): Open-source the adapter library to allow the community to contribute connectors for new brokers or specialized Thai financial products.
